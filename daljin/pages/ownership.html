권한 분리 매트릭스

인프라 항목 계정 명의 연구소 권한 에이전시 권한 비고
도메인 (daljin.com) 김달진미술연구소 ✅ 소유자 ❌ 접근 불가 도메인 이전·삭제 연구소만 가능
클라우드 Root 계정 김달진미술연구소 ✅ 소유자 ❌ 접근 불가 MFA 설정 필수
서버 (VM/컨테이너) 연구소 클라우드 계정 ✅ 전체 권한 ⚙️ 관리자 (IAM) 에이전시 IAM 계정 별도 생성
Managed DB 연구소 클라우드 계정 ✅ 전체 권한 ⚙️ 읽기/쓰기 DB Drop/Delete는 연구소 승인 필요
S3 오브젝트 스토리지 연구소 클라우드 계정 ✅ 전체 권한 ⚙️ 업로드/수정 버킷 삭제는 연구소만
CDN (Cloudflare) 김달진미술연구소 ✅ 전체 권한 ⚙️ Zone 관리자 결제 수단 연구소 카드
소스코드 저장소 (GitHub) 연구소 GitHub Org ✅ Owner ⚙️ Admin 납품 후 에이전시 계정 권한 회수
⛔ 절대 금지 사항
  • 클라우드 서버, DB, 도메인 등 어떠한 인프라도 에이전시 명의로 생성하는 행위
  • 에이전시가 Root/Master 계정 자격증명을 보유하거나 사용하는 행위
  • 계약 종료 후 에이전시에게 서버·DB 접근 권한이 잔존하는 상태 방치
  • DB 백업 파일, 이미지 원본 등 데이터를 에이전시 서버에 복사·보관하는 행위
✅ 계약 종료 시 이관 프로세스

개발 완료 및 계약 종료 시, 에이전시는 (1) 서버·DB 접근용 관리자 계정 삭제, (2) 소스코드 저장소 권한 회수, (3) SSH 키 폐기, (4) API 키/Secret 재발급을 즉시 이행해야 합니다. 이관 완료 확인서를 상호 서명 날인하여 보관합니다.
