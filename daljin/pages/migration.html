데이터 마이그레이션 전략

30년 이상 축적된 전시·작가·기사·박물관 데이터를 유실 없이 100% 새로운 DB 구조로 이관합니다.

100% 이관 필수 무손실 원칙 검증 보고서 제출

이관 대상 데이터 카테고리

🖼️
전시 정보
수만 건 이상
전시명 기간 장소 작가 관계망 이미지
개관전부터 현재까지의 전시 이력. 작가·갤러리·박물관과의 관계 데이터 포함. 기존 PHP 코드의 카테고리 분류 체계를 신규 taxonomy로 재매핑 필요.
👤
작가 프로필
수천 명 이상
작가명 약력 작품 이미지 전시 이력
한국 근현대 미술 작가 데이터베이스. 한글명·영문명 병기, 생몰연도, 학력·경력, 수상 이력 등 정형·비정형 데이터 혼재.
📰
기사·아카이브
수십만 건 이상
기사 본문 작성일 카테고리 첨부 이미지
뉴스·칼럼·리뷰·평론 등 텍스트 기반 아카이브. HTML 태그 정제 및 이미지 경로 S3로 일괄 변환 필요. 가장 볼륨이 큰 이관 대상.
🏛️
박물관·갤러리
수백 곳 이상
기관명 위치정보 운영시간 전시 연계
전국 미술관·갤러리·공공기관 정보. 전시와의 연계 관계 데이터 정합성 검증 필수. 지도 API 연동을 위한 좌표 데이터 보강 필요.

이관 5단계 프로세스

01
현행 DB 분석 및 스키마 문서화
기존 PHP/MySQL DB의 테이블 구조, 컬럼, 관계망, 데이터 타입을 전수 조사. 이상 데이터(NULL, 중복, 인코딩 오류) 목록 작성.
mysqldumpPython pandasERD 도구
02
신규 DB 스키마 설계 및 매핑표 작성
Laravel 마이그레이션 기반 신규 스키마 설계. 구 테이블 컬럼 → 신 테이블 컬럼 1:1 매핑표 문서화. 관계 재정의 (FK, 중간 테이블 등).
Laravel MigrationSpreadsheet 매핑표
03
이관 스크립트 개발 및 테스트 이관
PHP/Python 기반 자동화 이관 스크립트 개발. 스테이징 환경에 테스트 이관 실시 후 레코드 수·체크섬 일치 여부 검증.
PHP CLI ScriptPython ETLChecksum 검증
04
이미지 파일 S3 일괄 이전
기존 서버 디스크의 모든 이미지를 S3 오브젝트 스토리지로 이전. 기사·전시·작가 본문 내 이미지 경로(src) 일괄 치환. CDN URL로 재매핑.
AWS CLI / s3cmdURL 치환 스크립트셀레늄 검증
05
최종 검증 및 이관 보고서 제출
전체 레코드 수 대조, 이미지 누락 여부, URL 리다이렉트 동작 확인. 이관 검증 보고서 김달진미술연구소에 제출 및 서면 승인.
자동화 테스트 스위트이관 보고서 (Excel)

스키마 매핑 예시 (전시 테이블)

구 DB 컬럼명 신 DB 컬럼명 타입 변환 관계
bbs_exhibition.idx exhibitions.id INT → BIGINT UNSIGNED PK
bbs_exhibition.subject exhibitions.title VARCHAR(255) —
bbs_exhibition.artist_id exhibition_artist (pivot) INT → Many-to-Many N:M
bbs_exhibition.regdate exhibitions.created_at UNIX_TIMESTAMP → DATETIME —
bbs_exhibition.img_path media.s3_path 로컬 경로 → S3 URL FK
