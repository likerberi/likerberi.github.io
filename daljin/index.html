절대 원칙 요약

⛔ 하면 안 되는 것
  • 서버 하드디스크에 이미지 직접 저장
  • 인프라 계정 명의를 에이전시 명으로 생성
  • 기존 데이터 부분 이관 (100% 이관 필수)
  • CDN 없이 이미지 직접 서비스
✅ 반드시 해야 하는 것
  • 오브젝트 스토리지(S3) + Global CDN 구성
  • Managed DB (자동 백업 지원) 사용
  • 모든 계정 명의 = 김달진미술연구소
  • 제안서에 마이그레이션 방안·일정 명기
